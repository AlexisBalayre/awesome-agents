# Agents — Knowledge Hub

A curated knowledge base of **open-source tools, frameworks, and research** for LLM agents, kept as a single monorepo.

## Domains

| Domain | What |
|---|---|
| [Frameworks](./frameworks) | Agent frameworks & orchestration — incl. multi-agent coordination and role-based systems |
| [Protocols](./protocols) | MCP, A2A, tool calling & function calling — interop standards and integrations |
| [Memory](./memory) | Memory & state — long-term memory, context management |
| [Evals](./evals) | Agent benchmarks, evaluation harnesses, leaderboards |
| [Safety](./safety) | Guardrails, sandboxing, policy & permissions |

## How this is organized

Each domain has its own `README.md` with the same sections, so the structure folds in the
related sub-topics (specs, benchmarks, research) instead of scattering them:

- **Tools & Libraries** — open-source frameworks and toolkits
- **Protocols & Standards** — specs, schemas, interop conventions
- **Benchmarks & Leaderboards** — how agents are compared
- **Papers & Research** — key reading
- **Learning** — tutorials, courses, explainers

See [CONTRIBUTING.md](./CONTRIBUTING.md) for the entry format and curation rules.
